# Student Registration Portal

A Django-based Student Registration Portal developed using Python, Django, HTML, CSS, and SQLite. The application allows users to submit student information through a registration form and view registered student records in a structured table format.

## Features

- Student Registration Form
- Store Student Information in Database
- View Registered Students in a Table
- Success Confirmation Page
- SQLite Database Integration

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
